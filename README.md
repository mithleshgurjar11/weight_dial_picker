## 🚀 Usage

```dart
import 'package:weight_dial_picker/weight_dial_picker.dart';

WeightDialPicker(
  initialWeight: 70,
  unit: 'kg',
  onChanged: (value, unit) {
    print("$value $unit");
  },
)
```
